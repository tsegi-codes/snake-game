# Snake Game
This is a Snake Game built with Python

## How to play
* Use the **Up, Down, Left, and Right** arrow keys to control the snake.
* Eat the food (small blue circle) and increase your score.
* The game ends if you hit the wall or your own tail.

## Concepts Used
1. Python Turtle Graphics
2. Classes and Inheritance (OOP)
3. Lists slicing and Screen animation

## How to run the project
1. Install Python
2. Clone the repository
3. Run:
   ```bash
   python main.py
   ```
